Slider
Общие принципы работы элемента
    Компонент построен на основе препроцессоров Jade/Pug и SCSS, перед его использованием необходимо
необходимо подключить компонент через дерективу include( например так: include ../components/slider/slider),
к странице и через @import(например так: @import "./../components/slider/slider.scss";) к таблице стилей.
    При подключении компонента будет создана глобальная переменная aux_sliders которая содержит 
два массива items[] и views[] а также методы для взаимодействия с ними. Массив items[] хранит в себе 
объекты типа item(прототип объекта хранится aux_sliders.protos.item) объект item представляет данные 
для отдельного компонента на странице. Массив views[] содержит объекты view(прототип:aux_sliders.protos.view) 
хранящие ссылки на узлы DOM-дерева используемые для отображения данных.
... .

Как создать элемент
    1. Подключить необходимые файлы: slider.pug, slider.scss.
    2. Вызвать миксин +slider(args)
        args включают в себя:
            id: этот id будет присвоен базовому блоку компонента, объекту item, объекту view
            title: подпись над полосой бегунков
            pointCount: количество бегунков
                "one" - один, slider устанавливает одно значение
                "two" - два, slider определяет диапазон значений
            comment: подпись под полосой бегунков
            type: способ перемещения бегунков
                "solid" - бегуног может занимать произвольное положение, его значение расчитывается 
                        путем линейной интерполяции между двумя ближайшими значениями value миксина +slider_stage. 
                "step" - бегуног может занимать положение определяемое одним из элементов +slider_stage.
            name: это значение будет присвоено аттрибуту name скрытоuj элементf input(в котором хранится результат работы слайдера)
    3. Создать минимум два дочерних относительно +slider(arg) миксина +slider_stage(args)
        args включают в себя:
            index: опрделяет позицию точки(+slider_stage) относительно других +slider_stage.
            type: тип значения value, число либо строка.
            value: значение контрольной точки, число либо строка.
Как изменить состояние слайдера из внешнего кода
    Чтобы изменить состояние слайдера из внешнего кода неодходимо вызвать метод  aux_sliders.updateData(id, leftPosition, rigthPosition),
id - идентификатор слайдера корорый хотим изменить, leftPosition - левая граница слайдера в процентах,
rigthPosition - правая граница слайдера в процентах(например aux_sliders.updateData("firstSlider", 25, 75))ю
